# 📋 Task Mate - Gerenciador de Tarefas Moderno

Um gerenciador de tarefas elegant e poderoso construído com React, Vite, TypeScript e Tailwind CSS. Funcionalidades avançadas incluem sistema de prioridades, categorias, filtros inteligentes, dark mode e persistência automática em LocalStorage.

## ✨ Funcionalidades

### 📌 Principais
- ✅ **CRUD Completo** - Criar, ler, atualizar e deletar tarefas
- 🎯 **Sistema de Prioridades** - Alta, Média e Baixa com cores distintas
- 🏷️ **Categorias/Tags** - Organize suas tarefas por categoria
- 🔍 **Filtro de Busca** - Pesquise tarefas por título ou descrição
- 📅 **Data de Vencimento** - Adicione prazos às suas tarefas
- 🔄 **Ordenação** - Ordene por data, prioridade ou título

### 🎨 Design & UX
- 🌙 **Dark Mode** - Suporte completo a tema escuro
- 📱 **Responsivo** - Funciona perfeitamente em desktop, tablet e mobile
- ✨ **Animações Suaves** - Animações com Framer Motion
- 🎪 **Dashboard Visual** - Estatísticas em tempo real
- 🔔 **Ícones Lucide** - Ícones modernos e escaláveis

### 💾 Persistência
- 💾 **LocalStorage** - Seus dados são salvos automaticamente
- 🔄 **Sincronização em Tempo Real**
- 📊 **Backup Automático**

### 🏗️ Arquitetura
- ⚙️ **Context API** - Gerenciamento de estado centralizado
- 🪝 **Custom Hooks** - `useTaskManager` para fácil acesso
- 📝 **TypeScript** - Tipos seguros em todo o código
- 🧹 **Código Limpo** - Estrutura bem organizada

## 🚀 Quick Start

### Requisitos
- Node.js 16+
- npm ou yarn

### Instalação

```bash
# Instalar dependências
npm install

# Iniciar servidor de desenvolvimento
npm run dev
```

## 📖 Como Usar

- **Nova Tarefa**: Clique em "Nova Tarefa" e preencha os detalhes
- **Filtrar**: Use os filtros de categoria, prioridade, busca e status
- **Concluir**: Clique no checkbox para marcar como concluída
- **Editar**: Expanda a tarefa para ver mais opções
- **Dark Mode**: Clique na lua/sol para alterar tema

## 🌐 Tecnologias

- React 19
- Vite 8
- TypeScript 5.9
- Tailwind CSS 4
- Framer Motion 12
- Lucide React 1.7
